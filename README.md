军人工具箱微信小程序
============
##介绍

这是一个微信小程序，有4个页面：  
1. **【主页】**  
2. **【计算BMI指数】**  
3. **【资历章排列】**  
4. **【计算入伍天数和纪念日】**

## 使用方法
1. 打开微信
2. 进入小程序界面
3. 点击搜索框
4. 输入“军人工具箱”
5. 使用

## 运行修改方法
1. clone到本地
2. 使用微信开发者工具打开src文件夹即可调试运行
